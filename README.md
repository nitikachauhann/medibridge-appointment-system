# 🩺 MediBridge – Healthcare Appointment Platform

MediBridge is a modern web application designed to simplify doctor appointment booking and improve healthcare accessibility. It provides a seamless interface for users to browse doctors, view details, and manage appointments efficiently.

---

## 🚀 Features

* 🔍 Search doctors by specialization
* 📅 Book and manage appointments
* 👤 User-friendly interface
* ⚡ Fast and responsive UI

---

## 🛠️ Tech Stack

* **Frontend**: React.js (Vite + Tailwind CSS)
* **Backend**: Node.js, Express.js 
* **Database**: MongoDB 
* **Authentication**: JWT
